# 🌸 Muna - 女性健康管家

一个温暖的女性健康 App，帮助你记录和预测经期，跟踪情绪和身体症状，提供个性化的健康建议。

![Muna](https://img.shields.io/badge/version-1.0.0-purple)
![License](https://img.shields.io/badge/license-MIT-pink)

## ✨ 功能特性

- 📅 **经期记录与预测** - 智能算法预测下次经期
- 😊 **情绪追踪** - 每日心情打卡
- 🤒 **症状记录** - 记录身体不适症状
- ⚡ **能量水平** - 跟踪每日能量
- 🤖 **AI 建议** - 根据周期提供个性化建议
- 📊 **数据统计** - 查看历史记录和统计

## 🚀 如何部署

### GitHub Pages（免费）

1. 克隆仓库：
```bash
git clone https://github.com/yating-900/muna-app.git
cd muna-app
```

2. 推送代码：
```bash
git add .
git commit -m "Initial commit"
git push
```

3. 开启 GitHub Pages：
   - 进入仓库 → Settings → Pages
   - Source 选择 "main branch"
   - 保存

4. 访问你的网站：`https://yating-900.github.io/muna-app/`

## 📱 使用方法

1. 打开 App
2. 在首页点击"记录经期"
3. 每日打卡记录心情、能量和症状
4. 查看日历了解经期历史和预测

## 🛠 技术栈

- 纯 HTML/CSS/JavaScript（单文件应用）
- LocalStorage 本地数据存储
- 响应式设计，适配手机和桌面
