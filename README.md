# Muna - 女性健康管家

一个安静、克制的女性健康应用，记录自然节律与自我觉察。

![Muna](https://img.shields.io/badge/version-1.0.0-brightgreen)
![License](https://img.shields.io/badge/license-MIT-green)

## 理念

Muna 是一个不同于传统经期 App 的存在。
不是"可爱少女风"，而是"记录自然节律与自我觉察的安静空间"。

## 风格

- 日式极简设计
- 温润米色系统
- 纸张质感
- 充足留白
- 安静高级

## 功能

- 经期记录与智能预测
- 情绪追踪
- 身体症状记录
- 能量水平跟踪
- AI 个性化建议
- 数据统计
